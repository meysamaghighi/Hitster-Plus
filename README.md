# Hitster Plus

A web app that generates bonus questions for the Hitster board game.

## Setup

1. Add your songs to `songs.js`
2. Push to GitHub
3. Enable GitHub Pages in repository settings
4. Access at: `https://meysamaghighi.github.io/Hitster-Plus/`

## Usage

1. Select a song from the dropdown
2. Click "Show questions" 
3. Display questions to players during the game
